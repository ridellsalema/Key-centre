# Key Centre & Mobile Locksmith

This is a simple, professional landing page for Key Centre & Mobile Locksmith.
It is designed to be lightweight, mobile-friendly, and easy to deploy.

## Features

- **Mobile First Design**: Works perfectly on smartphones.
- **Direct Contact Actions**: One-tap Call and WhatsApp buttons.
- **Clean UI**: Professional look matching the business identity.

## How to Run Locally

1. Clone the repository.
2. Open `index.html` in your browser.

## Deployment

This project is ready to be deployed on GitHub Pages.

1. Go to Settings > Pages.
2. Select `main` branch and `/root` folder.
3. Save.
